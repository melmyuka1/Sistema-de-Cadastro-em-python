Este projeto é uma versão reorganizada do sistema  
desenvolvido anteriormente. Agora, o código está modularizado
 e melhor estruturado, com arquivos separados para cada funcionalidade
  principal do sistema, e todos os dados salvos em arquivos .json.

  O projeto anterior está comentado mais ainda presente e ele seria o arquivo ListaEstudantes.py

## ✅ Funcionalidades

- Cadastro, listagem, edição e remoção (CRUD) de:
  - Estudantes
  - Professores
  - Disciplinas
  - Turmas
  - Matrículas

## 📦 Requisitos

- Python 3.7+
- Nenhuma biblioteca externa é necessária (uso apenas da biblioteca `json`)
